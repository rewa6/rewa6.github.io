# Data Engineer Portfolio

A clean, dark-themed portfolio website built with HTML, CSS, and vanilla JavaScript.  
Hosted on GitHub Pages.

## 🚀 Live Site
[https://yourusername.github.io](https://yourusername.github.io)

## 🛠 Stack
- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Syne + DM Mono)
- Canvas API for particle animation
- IntersectionObserver for scroll reveals
- GitHub Pages for hosting

## 📁 Structure
```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── assets/
│   └── resume.pdf      ← add your resume here
└── README.md
```

## ✏️ Customise
1. Replace `Your Name` in `index.html`
2. Update email, GitHub, LinkedIn links
3. Edit the projects, experience, and stats sections
4. Drop your `resume.pdf` in `assets/`
5. Change `YN` in the nav logo to your initials

## 📦 Deploy to GitHub Pages
See the deployment guide in the project README or follow the steps below:

1. Create a repo named `yourusername.github.io`
2. Push all files to the `main` branch
3. Go to Settings → Pages → set source to `main` branch / root
4. Your site is live in ~1 minute!
